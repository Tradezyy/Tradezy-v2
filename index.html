How auto-loading works

When deployed to GitHub Pages or Render, this page automatically fetches data/closed_trades.csv from the same folder as index.html. To refresh your dashboard, just commit a new version of that file and push. GitHub Pages picks it up within about a minute — your friend simply reloads the page to see fresh numbers. The CSV upload above is a fallback for testing or when you want to inspect a different file in the browser.
